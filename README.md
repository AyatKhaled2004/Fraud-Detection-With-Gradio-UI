
# 💳 Credit Card Fraud Detection

## 📌 Overview
This project focuses on detecting fraudulent credit card transactions using machine learning techniques.

The goal is to build a model that can distinguish between normal and fraudulent transactions, helping reduce financial risks and improve security.

---

## ⚙️ How it works
The system takes transaction data as input and predicts whether it is:

- ✅ Normal transaction  
- ❌ Fraudulent transaction  

The model is trained on real-world transaction data and learns patterns that indicate fraud.

---

## 🧠 Techniques Used
- Data preprocessing  
- Handling imbalanced datasets  
- Feature analysis  
- Machine Learning models for classification  

---

## 🛠️ Technologies
- Python  
- Pandas  
- NumPy  
- Scikit-learn  

---

## 📊 Dataset

This project uses the **Credit Card Fraud Detection Dataset** from Kaggle:

🔗 https://www.kaggle.com/datasets/mlg-ulb/creditcardfraud  

### Dataset features:
- Real credit card transactions  
- Highly imbalanced data (fraud cases are rare)  
- Features transformed using PCA  
- Includes important fields like `Time` and `Amount`
